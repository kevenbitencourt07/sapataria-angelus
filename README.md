# Sapataria Angelu's

Site institucional da **Sapataria Angelu's**, uma sapataria artesanal dedicada ao reparo de peças e artigos de luxo.

# Sobre o projeto

Este projeto foi desenvolvido para apresentar a marca **Angelu's**, destacando seus serviços, valores e o processo artesanal de reparo.

O objetivo é oferecer uma presença online profissional, permitindo que clientes conheçam os serviços e entrem em contato com facilidade.

# Funcionalidades

- Apresentação da sapataria
- Catálogo de serviços
- Informações de contato
- Layout responsivo (adaptado para celular e desktop)

# Tecnologias utilizadas

- HTML
- CSS
- JavaScript
